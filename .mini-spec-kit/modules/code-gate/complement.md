# Code Gate Complement

## Responsibilities

- Define hard, local, file-based phase gates for mini-spec-kit.
- Keep business verification in `checklist.md` and workflow state in `gate.md`.
- Keep logs lightweight with `verify.log` and `gate-history.log`.

## Non-Goals

- No services, databases, background agents, or deployment systems.
- No replacement of the six-phase workflow.
